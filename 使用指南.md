## 通用准备

每位使用者都应在自己的电脑上创建一个虚拟环境。它只保存本项目需要的 Python 库，不包含业务数据、登录信息或数据库内容：

```powershell
cd "C:\Users\haoxi\Desktop\作业"
py -3 -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

以后每次新开 PowerShell，要先执行：

```powershell
cd "C:\Users\haoxi\Desktop\作业"
.\.venv\Scripts\Activate.ps1
```

需要使用牛客的浏览器登录或接口发现功能时，再安装浏览器内核：

```powershell
python -m playwright install chromium
```

虚拟环境不是数据库。只有指南中明确标注“需 MySQL”的项目才需要安装、启动并配置 MySQL；牛客采集工具不需要数据库。

## 4. 牛客网题库采集工具

主入口：[nk_api_collector.py](牛客网题库采集工具/nk_api_collector.py)。该项目的详细说明在其 [README.md](牛客网题库采集工具/README.md)。

该工具不连接 MySQL。首次运行只需在打开的浏览器中使用**使用者自己的牛客账号**登录，采集结果将直接写入 JSON 文件。

```powershell
cd "C:\Users\haoxi\Desktop\作业\牛客网题库采集工具"
python nk_api_collector.py --papers 1
```

首次运行会要求在浏览器登录。登录后回终端按回车，程序把会话保存在 `config/session.json`，随后将结果写到 `output/`。`session.json` 含登录信息，不能发送或提交。

常用命令：

```powershell
# 指定知识点
python nk_api_collector.py --papers 10 --knowledge Java

# 采集指定试卷 URL
python nk_manual.py "https://www.nowcoder.com/exam/test/123456/detail"

# 从导出的 Cookie 文件更新会话
python nk_session_refresh.py --cookie-json "C:\path\to\nowcoder-cookies.json"
```

`nk_api_discover.py` 会直接启动浏览器进行接口发现，不支持 `--help`；只在接口变更、主采集器无法使用时再运行它。

## 7. 登录注册系统（需 MySQL）

入口：[app3.py](登录注册系统/app3.py)。在启动 PowerShell 中设置密钥和数据库连接信息：

```powershell
cd "C:\Users\haoxi\Desktop\作业\登录注册系统"
$env:SECRET_KEY = "请替换为一段随机且保密的长字符串"
$env:DB_HOST = "localhost"
$env:DB_PORT = "3306"
$env:DB_USER = "root"
$env:DB_PASSWORD = "你的数据库密码"
$env:DB_NAME = "你的数据库名"
python app3.py
```

访问 `http://127.0.0.1:5000`。首次启动会自动建 `users` 表。请勿把真实 `SECRET_KEY`、数据库密码或牛客 `session.json` 写入代码或发给他人。
